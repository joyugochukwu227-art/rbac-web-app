RBAC Web Application

Description
This is a simple Role-Based Access Control (RBAC) web application built using Flask. It demonstrates basic authentication and authorization by allowing users to log in and access pages based on their assigned roles (Admin or User).


Features
- User login system
- Role-based access control (Admin / User)
- Session-based authentication
- Protected routes
- Simple and lightweight design

 How It Works
- Users log in with their credentials
- The system checks their role
- Admin users can access admin page
- Regular users have limited access
- Unauthorized access is blocked
